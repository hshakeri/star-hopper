# Star Hopper Project Roadmap 🗺️

Star Hopper is a community-driven, open-source project. Our mission is to make physics and computer programming education accessible, interactive, and fun for children worldwide.

To keep this project 100% free of ads, tracking, and paywalls, we rely entirely on open-source sponsorships and community donations. Below are our development phases and funding milestones.

---

## 🌟 Current Version: Space Cadet Release (v1.0)
* **Status**: **Active / Live**
* **Features Included**:
  * Five interactive planetary worlds (Earth, Moon, Jupiter, Glacies, Magnet-V).
  * Real-time scientific telemetry: Mechanical Energy charts ($KE$, $PE$, $TE$), velocity vector lines, and friction index scales.
  * Sandbox KidCode Terminal: Friendly compiler parsing variables, loops, conditional blocks, object spawning, and error debugging.
  * Spaceship Orbital Navigation Mode using high-accuracy Velocity Verlet integration.
  * Science Journal notebook with reflection logs and Space Academy graduation certificates.

---

## 🚀 Funding Milestones & Future Features

### Milestone 1: The Space Builder Update 🛠️
* **Goal**: **$500 / month** in GitHub Sponsors or Buy Me a Coffee donations.
* **Features to Build**:
  * **Interactive Level Creator**: Allow students to drag-and-drop tiles, portals, magnetic nodes, and custom friction zones.
  * **Level Serialization**: Export custom levels as shareable JSON strings so students can challenge their classmates.

### Milestone 2: Native Python & Javascript Compiler 🐍
* **Goal**: **$1,500 / month** in community sponsorships.
* **Features to Build**:
  * **Advanced Code Editor**: Replace the simple command bar with a side-by-side Monaco or Ace code editor.
  * **Strict Syntax Matching**: Allow students to toggle between KidCode (simplified), standard Python, and standard JavaScript APIs to bridge the gap to real-world software engineering.

### Milestone 3: Cooperative Space Flight Command 🛰️
* **Goal**: **$3,000 / month** in community sponsorships.
* **Features to Build**:
  * **Multiplayer/Co-op Terminals**: Allow two students to control different parts of the physics engine concurrently (e.g. one controls ship navigation, the other programs planet gravity to assist).
  * **Global Sandbox Repository**: A secure, moderated community hub for sharing levels and custom scripts.
